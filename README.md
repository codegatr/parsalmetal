# Parsal Metal Alüminyum CMS

**Versiyon:** 1.0.0  
**PHP:** 8.3+  
**Veritabanı:** MySQL 5.7+ / MariaDB 10.4+  
**Tablo Prefix:** `prs_`  
**Geliştirici:** CODEGA (codega.com.tr)

---

## Kurulum

1. Dosyaları sunucuya yükleyin
2. `/install/` adresini açın
3. Adımları takip edin

---

## Admin Panel

`/admin/` — Admin giriş sayfası

---

## Güncelleme

Admin panel → Güncelleme sayfasından GitHub Releases üzerinden tek tıkla güncelleme yapılabilir.

---

## Dizin Yapısı

```
/install/       → Kurulum sihirbazı
/includes/      → Core PHP dosyaları
/pages/         → Frontend sayfa dosyaları
/admin/         → Admin panel
/assets/        → CSS, JS, görseller
/update/        → Güncelleme sistemi
config.php      → Veritabanı ayarları (otomatik oluşturulur)
```
